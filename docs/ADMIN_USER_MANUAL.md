# Admin User Manual

## Login

1. Open `/admin/login`.
2. Enter your email and password.
3. Click **Login**.

Sample Super Admin:

```text
admin@marymother.local
admin123
```

## Dashboard

The dashboard shows:

- total legacy posts
- total caregiver stories
- total gallery images
- total contact messages
- total published content
- recent activity logs

## Manage Pages

Use **Pages** to edit text for:

- Home
- About
- Legacy
- Support
- Contact

Open a page, update the title or content, choose Draft or Published, then save.

## Manage Legacy Content

Use **Legacy Content** to create, edit, publish, or delete foundation milestones.

Recommended content:

- history
- mission
- values
- service milestones
- community impact

Avoid private personal stories.

## Manage Caregiver Stories

Use **Caregiver Stories** to add reflections from nurses, caregivers, staff, or volunteers.

Privacy rule:

```text
Do not include real resident names, private medical details, personal trauma, or identifiable information unless written consent is documented.
```

Good story topics:

- lessons learned from service
- teamwork
- compassion
- daily care routines
- why staff continue serving

## Manage Gallery

Use **Gallery** to:

- add categories
- upload images
- edit captions
- update image status
- delete images

Use respectful images and avoid private resident-identifying photos unless the foundation has documented consent.

## Manage Events

Use **Events** to:

- create volunteer orientations
- create donation drives
- publish community activities
- review participation requests
- mark requests Pending, Confirmed, or Declined

Participants submit requests from the public Events page. Staff should coordinate before confirming visits or donation drop-offs.

## Manage Website Videos

Use **Videos** to add approved YouTube or Vimeo links.

Recommended flow:

1. Upload the approved video to YouTube or Vimeo.
2. Copy the public or unlisted video link.
3. Paste the link into the CMS.
4. Publish the video.

The website embeds the video and also gives visitors a direct external watch link.

## View Donation Records

Use **Donations** to review PayMongo donation checkout records.

Statuses:

- Pending
- Redirected
- Paid
- Cancelled
- Failed

For real donations, confirm final payment status in the PayMongo dashboard or through the configured webhook.

## Manage Support Info

Use **Support Info** to update:

- donation instructions
- bank details
- in-kind donation needs
- contact person
- contact number
- foundation address
- foundation email
- Facebook URL
- YouTube URL
- Google Maps query

Online donations use PayMongo Hosted Checkout when PayMongo keys are configured.

## Manage Messages

Use **Messages** to view public contact form submissions.

Content Editors and Super Admins can mark messages read/unread or delete them.

Viewers can only read messages.

## Manage Admin Users

Super Admins can:

- add users
- edit users
- change roles
- deactivate users

Roles:

- Super Admin
- Content Editor
- Viewer

## Logout

Click **Logout** in the sidebar when finished.
